# Chemostat Competition Time-Series

Time-series analysis and visualization of co-culture competition between CCR strategies in a chemostat with periodic substrate switching.

This module loads initial conditions from independent cultivation results, runs the competition simulation for a long period until periodic dynamics are established, then extracts the last complete cycle for detailed visualization (P1-P5) and QuadGK numerical integration analysis of synthesis rates and resource investment ratio.

## Plots

- **P1**: Biomass dynamics — long-term competition trajectory showing biomass of both strategies over the full simulation.
- **P2**: All state variables within a single period — carbon source, mRNA, enzymes, intracellular substrate, intracellular resource, and biomass for each strategy during the last complete cycle.
- **P3**: Variable differences between strategies — difference of each variable (strategy 1 minus strategy 2), highlighting which strategy has higher values and when.
- **P4**: Specific growth rate — total specific growth rate (rg) and decomposed into contributions from S1 pathway (rg1) and S2 pathway (rg2) for each strategy.
- **P5**: Specifc growth rate differences — difference in rg, rg1, and rg2 between strategies, revealing which pathway contributes to the comparative advantage.

## QuadGK Integration Analysis

QuadGK numerical integration over the last complete cycle computes the following for each strategy:

- **Synthesis rates**: mRNA synthesis rate and enzyme synthesis rate integrals for each pathway, representing the total resource cost for building pathway machinery over one cycle.
- **Resource investment ratio P1**: proportion of total synthesis allocated to the S1 pathway, quantifying the resource allocation strategy.

## Files

- **`competition_chemostat_timeseries.jl`** — Self-contained module including: parameter struct and defaults, ODE system, S1 pulsed inflow function, initial condition loading from independent cultivation CSV, two-stage simulation (long-term + dense output last cycle), QuadGK integration analysis, and P1-P5 visualization functions.

## Initial Conditions

Initial conditions are loaded from independent cultivation summary CSVs located in the `initial_data/` subdirectory. The expected filename format is:

```
initial_data/independent_grid_combined_summary_D_<D>_<strategy_pair>.csv
```
These files are generated by the Independent_chemostat.

If the file is not found, fallback initial conditions defined in `main()` are used.

## Usage

1. Place the independent cultivation summary CSV in the `initial_data/` subdirectory
2. Edit `main()` to set:
   - `strategy_indices`: pair of strategies to compare (e.g., `[0,1]` or `[1,2]`)
   - `target_tau`: simulation length in generations
   - Parameters in `Parameters()` constructor (D, T1, T2, etc.)
3. Run in Julia/VSCode: `include("competition_chemostat_timeseries.jl")`
4. Plots are displayed and saved as PDF to `plot_results/`
